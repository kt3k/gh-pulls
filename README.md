> CLI tool for showing numbers of pull requests by the user of the given user name.

# Usage

Install via npm:

```console
$ npm i -g gh-pulls
```

Hit the command `gh-pulls` with the names.

```console
$ gh-pulls mojombo
mojombo: 43
```
